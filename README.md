# Snake Game (Python)

This is a classic Snake Game built using Python and the Turtle graphics library.

## Features
- Snake movement using keyboard controls
- Food generation
- Score tracking
- Highest score tracking

## Technologies Used
- Python
- Turtle Graphics

## How to Run
1. Install Python
2. Download the project
3. Run the file:

python snake_game.py
